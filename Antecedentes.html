Antecedentes


Para realizar ciertas partes del código, que teníamos duda o que simplemente no conocíamos o estábamos aprendiendo, al estar investigando encontramos por internet otros repositorios de GitHub y otros códigos que hablaban de lo mismo o con temas parecidos que nos ayudarías con la elaboración y seguimiento del nuestro.

Principalmente, comenzando por el repositorio del maestro, ya que con eso fue con el que estuvimos trabajando al principio, donde aprendimos los conceptos básicos del de sarrollo del código en todos los aspectos. Fue el código que la mayoría de nuestros compañeros de la clase se basaron y el que nos ayudo mucho en nuestro proyecto.

C. Cenado. (2018). css/estilos.css. 07 de septiembre del 2019, de GitHub Sitio web: https://github.com/Magtimus/boton-efecto-ripple-waves

De este otro repositorio se utilizó para los efectos de botón de la carpeta css en style, el cómo hacer que este tenga un degradado, los bordes redondos una transición hacia abajo con una sobra que lo siguiera para darle estética a estos.

G.Yoshev. (2019). animate.css. 06 de septiembre del 2019, de GitHub Sitio web: https://github.com/daneden/animate.css

También de este repositorio obtuvimos la manera de usar las animaciones, ya que cada uno presentaba las distintas formas de las que se podían utilizar, ya sea descargándolo, colocando el link en el index, y a la vez ejemplos de cómo colocarlo en el css.

Por otra parte, el resto de los repositorios solo fueron tomados como base, como ejemplos para poder llevar a cabo el proyecto de la mejor forma, lo mismo ocurrió con W3school, de donde sacamos ejemplos que fueron tomados como base, y el uso correcto de cada parte desconocida del código.
