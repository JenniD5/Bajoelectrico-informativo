¿Cómo se usa?


El usuario puede escuchar la nota perfectamente entonada, para usar simplemente haz clic en la cuerda que deseas afinar o da clic a las teclas “q” para Sol, “w” para Re, ”e” para La, ”r” para Mi.

Presta atención a la altura del sonido: si estiras la cuerda e l sonido será más agudo, si la relajas el sonido será más grave.
